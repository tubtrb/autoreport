# v0.4 Design Notes

Use this directory for branch-local framing docs that explain how `incubator2`
extends the current `autoreport_editorial` engine without changing public
runtime behavior.

Each design note should answer:

- which current `v0.3` surface it depends on
- whether the artifact is `carry_as_is`, `carry_retarget`, or `defer`
- what validation evidence is required before promotion
- why the work still belongs in `experiments/v04/`

Start from `v0.4-roadmap.md` when deciding what to work on next.
